Lesson 0029

求值并打印字符串字面量

scanner 认识字符串、parser 认识字符串。这一课只做 evaluator / runtime：让字符串真正变成运行时值，并被 print 输出。

这一步为什么重要： 字符串走到这一课才算真正"活了"。在此之前它只是 token 和 AST 节点，不能运行也不能输出。这一课之后，print "hello" 就能真正看到结果。

本课范围

  • 给 Value 新增字符串值变体
  • 在 eval_expr 中为 Expr::String 添加求值分支
  • 在 print 语句中支持输出字符串值
  • 新增 evaluator 测试

这一课不做：

  • 不做字符串运算
  • 不做字符串比较
  • 不做字符串 pattern
  • 不改 scanner
  • 不改 parser / AST

这节课真正新增的知识点

一个新字面量进入语言，最终要走过一条完整路径：

scanner (token) -> parser (AST) -> evaluator (runtime value)

第 27 课做了 scanner，第 28 课做了 parser，这一课就做最后一环：把字符串字面量变成运行时值。

推荐的最小实现方向

  1. 在 Value 中加入字符串变体
  2. 在 eval_expr 的 Expr::String(s) 分支返回对应运行时值
  3. 在 print 的输出 match 中增加字符串打印分支
  4. 新增 evaluator 测试，验证完整链路

目标输入 / 输出

输入:
print "hello";

期望输出:
hello
输入:
let s = "world";
print s;

期望输出:
world
输入:
let name = "copilot";
print name;

期望输出:
copilot

完成标准
