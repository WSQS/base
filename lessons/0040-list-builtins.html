Lesson 0040

list 操作 builtin

list 已经可以作为值存储和打印。这一课只做 evaluator：注入四个 list 操作 builtin，让 list 第一次可以真正被操作。

为什么重要： 没有操作的 list 只是一个静态容器。len/head/tail/push 四个操作覆盖了自举中最基本的遍历、取元素和构造。

本课范围

  • 实现四个 builtin：len、head、tail、push
  • 在 eval_program() 中注入到初始环境
  • 新增 evaluator 测试

这一课不做：

  • 不改 scanner / parser / AST
  • 不做 list pattern 或 list 高阶函数

四个 builtin

len(list) → Integer
返回 list 的长度。len([1, 2, 3]) → 3
head(list) → Value
返回 list 的第一个元素。head([]) 报错或返回默认值
tail(list) → List
返回去掉第一个元素后的新 list。tail([1, 2, 3]) → [2, 3]
push(list, x) → List
返回在后面追加 x 的新 list。push([1, 2], 3) → [1, 2, 3]

目标输入 / 输出

let x = [1, 2, 3];
print(len(x));   → 3

let y = tail(x);
print(y);        → [2, 3]

let z = push([1, 2], 3);
print(z);        → [1, 2, 3]

完成标准

  • len、head、tail、push 四个 builtin 均已实现
  • 全部在 eval_program() 中注入
  • 新增 evaluator 测试并通过 cargo test
  • scanner / parser / AST 未改动
提醒： 四个 builtin 都是同一个模式——match Value::List → 调 Vec 方法 → 返回 Value。
