Lesson 0027

扫描字符串字面量

这一课只做 scanner：让语言第一次认识双引号包裹的字符串字面量，并把它们产出为单个 token。

这一步为什么重要： 你现在的语言已经能处理整数、布尔值和最小 match。加入字符串的第一步，不是立刻支持 parser 和 evaluator，而是先让 scanner 能正确把 "hello" 这种输入视为一个完整单位。

本课范围

  • 给 Token 新增字符串 token 变体
  • 当 scanner 看到 " 时，读取直到下一个 " 为止
  • 把中间内容作为一个完整字符串 token 推入结果
  • 新增 scanner 测试

这一课不做：

  • 不改 parser
  • 不改 AST
  • 不改 evaluator
  • 不支持转义（如 \"、\n）
  • 不支持多行字符串
  • 不支持字符串运算或字符串 pattern

这节课真正新增的知识点

scanner 不只是把单字符拆开；它也负责识别跨多个字符的单个词法单元。

"hello" 不是 7 个彼此独立的字符，而是 1 个字符串 token。

也就是说，scanner 需要在遇到起始双引号后，切换到“读取整个字符串内容”的模式，而不是继续按普通字符逐个分发。

第一版字符串规则

  • 形式："任意非双引号内容"
  • 字符串内容原样收集
  • 直到遇到下一个 " 才结束

例如下面这些都应被视为字符串字面量：

"hello"
"abc123"
"true"
"1 + 2"

注意："true" 在这一课里仍然是字符串，不是布尔关键字。

推荐的最小实现方向

  1. 先在 Token 中加入 String(String)
  2. 在 scan() 里遇到 " 时，先 flush()
  3. 新建一个临时字符串缓冲区
  4. 持续读取字符，直到再次遇到 "
  5. 把收集到的内容包装为 Token::String(...)
  6. 新增测试验证 scanner 结果

目标输入 / token 输出

输入:
print "hello";

目标 token:
[
  Print,
  String("hello"),
  Semicolon,
]
输入:
"true" "123"

目标 token:
[
  String("true"),
  String("123"),
]

完成标准

  • scanner 已支持读取最小 "..." 形式的字符串字面量
