Lesson 0006

让扫描器第一次识别名字：标识符

这一课只教一件事：让你的扫描器不再只认识数字和符号，而是第一次识别由字母组成的“名字”。

和你的 mission 的关系： 你想实现自己的语言，而真实语言一定不只有数字和运算符，还要有变量名、函数名、关键字。今天是你第一次让扫描器进入“名字语言”阶段，这会让后续的 let x = 1; 变得可能。

今天只新增这一类 token

新增：

  • Ident(String)

暂时还不做：

  • 关键字区分（例如 let、print）
  • 数字和字母混合规则的复杂情况
  • 完整错误处理

本课目标输入 / 输出

输入: foo = 12;
输出: Ident("foo")
      Equal
      Number(12)
      Semicolon

输入: (bar+3)
输出: LParen
      Ident("bar")
      Plus
      Number(3)
      RParen

也就是说，扫描器要学会把连续字母读成一个整体，而不是把它们当作不支持字符逐个打印出来。

你要亲手完成的步骤

  1. 在 enum Token 中新增 Ident(String)
  2. 给扫描器增加“累计名字”的逻辑，就像你之前累计数字一样
  3. 当遇到字母时，持续往后读取，直到不是字母为止
  4. 把读到的整段名字推入 Ident(...) token
  5. 补充单元测试，验证名字与已有 token 混排时的输出顺序

卡住时优先检查这 3 件事

  • 是不是把标识符按单个字符切碎了，而没有持续读取整个名字？
  • 遇到字母后，结束条件是否写对了？
  • 名字 token 与已有的数字 / 符号 token 混排时，顺序是否正确？
本课后的观察点： 完成这一课后，你的扫描器第一次具备了“识别名字”的能力。下一步只要再加关键字区分，像 let x = 1; 这样的真实语句就开始成形了。
