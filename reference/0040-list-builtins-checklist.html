Reference 0040

list builtins 清单

用途：快速确认 list 操作 builtin 已全部实现并注入。

本课后应做到

  • len、head、tail、push 均已实现
  • 全部在 eval_program() 中注入
  • 新增测试并通过

四个 builtin 速查

len(list) → Integer      // 返回长度
head(list) → Value       // 返回第一个元素
tail(list) → List        // 返回去掉第一个元素后的新 list
push(list, x) → List     // 在后面追加 x

最小验证样例

len([1, 2, 3]) → 3
head([1, 2, 3]) → 1
tail([1, 2, 3]) → [2, 3]
push([1, 2], 3) → [1, 2, 3]

最容易犯的错误

  • 实现了 builtin 但忘记在 eval_program() 中注入
  • push 的参数顺序写反
  • tail/head 对空 list 没有防护
