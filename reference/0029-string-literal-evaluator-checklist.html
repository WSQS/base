Reference 0029

字符串字面量 evaluator 清单

用途：快速确认 evaluator / runtime 是否已经支持字符串值并能被 print 输出。

本课后应做到

  • Value 中新增了字符串运行时值
  • eval_expr(Expr::String(s)) 返回对应运行时值
  • print 已支持输出字符串
  • 新增 evaluator 测试并通过

当前仍不要求

  • 字符串运算
  • 字符串比较
  • 字符串 pattern

最小验证样例

输入:
print "hello";

期望输出:
hello
输入:
let s = "world";
print s;

期望输出:
world

最容易犯的错误

  • 新增了 Value::String，但忘记在 eval_expr 中处理 Expr::String
  • 新增了 eval_expr 分支，但 print 的 match 分支没有更新，导致输出时跳过字符串
  • 一边做 evaluator，一边顺手给字符串加运算或 pattern，把课题做大
