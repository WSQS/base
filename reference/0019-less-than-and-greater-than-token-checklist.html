Reference 0019

< 和 > token 清单

用途：快速确认 scanner 是否已经支持最小比较符号 < 与 >。

本课后应做到

  • Token 中有表示 < 和 > 的变体
  • scan() 能识别这两个单字符运算符
  • 新增 scanner 测试并通过

当前仍不要求

  • parser 理解比较表达式
  • evaluator 计算布尔结果
  • == 与 != 的双字符扫描

最小验证样例

let tokens = scan("1 < 2 > 3");

目标 token 序列：
[Number(1), Less, Number(2), Greater, Number(3)]

最容易犯的错误

  • 只改了测试，忘了在 Token 中新增变体
  • 只改了 Token，忘了在 scan() 中处理字符分支
  • 提前去改 parser / evaluator，把这一课做大
