Reference 0027

字符串字面量 scanner 清单

用途：快速确认 scanner 是否已经支持最小字符串字面量。

本课后应做到

  • Token 中新增了字符串 token 变体
  • scanner 看到 " 时会读取直到下一个 "
  • 中间内容会被收集成一个完整字符串 token
  • 新增 scanner 测试并通过

当前仍不要求

  • parser 支持字符串
  • AST / evaluator 支持字符串
  • 转义（\"、\n）
  • 多行字符串
  • 字符串运算
  • 字符串 pattern

最小验证样例

输入:
print "hello";

目标 token:
[
  Print,
  String("hello"),
  Semicolon,
]
输入:
"true" "123"

目标 token:
[
  String("true"),
  String("123"),
]

最容易犯的错误

  • 没有先 flush()，导致前一个标识符/数字污染字符串读取
  • 读取到字符串内容后忘记跳过结束双引号
  • 把 "true" 误当成关键字 true
  • 一边做 scanner，一边顺手改 parser / evaluator，把课题做大
