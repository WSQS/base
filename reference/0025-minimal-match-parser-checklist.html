Reference 0025

最小 match parser 清单

用途：快速确认 parser 是否已经支持最小 match 表达式外壳，并引入了独立的模式 AST。

本课后应做到

  • AST 中新增了独立的 Pattern
  • AST 中新增了最小 MatchArm 与 Expr::Match（或等价结构）
  • parser 已支持 match expr { pattern => expr, ... } 外壳
  • 第一版模式仅支持：true、false、整数字面量、_
  • 新增 parser / AST 测试并通过

当前仍不要求

  • match 求值执行
  • 变量绑定模式
  • guard
  • 穷尽性检查
  • 多语句 arm

最小验证样例

print match true {
    true => 1,
    false => 0,
    _ => 2,
};

目标：
得到 Expr::Match（或等价节点）
- subject = Boolean(true)
- arms = [true => 1, false => 0, _ => 2]

最容易犯的错误

  • 偷懒复用 Expr 表示模式，后续会与绑定语义冲突
  • 一边做 parser，一边顺手开始做 evaluator，把课题做大
  • 测试只验证“能 parse”，却不验证 Pattern 和 arm 结构是否正确
