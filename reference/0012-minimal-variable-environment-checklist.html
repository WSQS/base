Reference 0012

最小变量环境清单

用途：在解释器第一次支持变量存取时，快速确认你是否已经让程序具备最小状态能力。

本课后应做到

  • 有一个最小环境，如 HashMap<String, i64>
  • let 能把值写入环境
  • Ident 能从环境读取值
  • let x = 1 + 2; print x; 能输出 3

当前仍不要求

  • 嵌套作用域
  • 变量重新赋值
  • 未定义变量的完整错误系统
  • 更复杂的运行时对象

最小思路

eval_program(program)
  let mut env = HashMap::new()
  遍历每条 stmt

  let name = expr;
    先算 expr 的值
    再 env.insert(name, value)

  print expr;
    先 eval_expr(expr, &env)
    再输出结果

 eval_expr(expr, env)
   Number(n) => n
   Ident(name) => 从 env 取值
   Binary(left, op, right) =>
     递归计算左右两边

验证样例

  • let x = 1 + 2; print x; → 输出 3
  • let a = 8; print a; → 输出 8

最容易犯的 3 个错误

  • 把表达式结构存进环境，而不是把表达式求值结果存进去
  • 递归求值 binary 时没有继续把环境传下去
  • 把环境放在错误的位置，导致后续语句看不到前面的变量
