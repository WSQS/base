Reference 0006

标识符扫描清单

用途：在现有扫描器上第一次加入 Ident(String) 时，快速检查你是否正确处理了“连续名字字符”。

本课后应支持

  • Ident(String)
  • 名字与数字、符号混排
  • 整段名字一次性读出

当前仍不做

  • 关键字区分
  • 复杂命名规则
  • 多字符运算符
  • 完整错误恢复

识别标识符的最小流程

1. 遇到字母，开始读取名字
2. 持续向后读，直到不是字母为止
3. 把整段名字输出为 Ident(...)
4. 继续扫描后续 token

验证样例

  • "foo = 12;" → Ident("foo"), Equal, Number(12), Semicolon
  • "(bar+3)" → LParen, Ident("bar"), Plus, Number(3), RParen
  • "x+y" → 两个 Ident 和一个 Plus

最容易犯的 3 个错误

  • 把 foo 错误地拆成 f、o、o
  • 名字扫描结束条件写错，导致���后面的符号吃进去
  • 名字和数字 / 符号混排时，token 顺序错误
