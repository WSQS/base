Reference 0002

NUMBER 扫描清单

用途：实现只支持数字的最小扫描器时，快速检查你是否漏了关键步骤。

最小目标

  • 输入是字符串
  • 扫描器从左到右前进
  • 遇到数字时持续读取整个数字片段
  • 输出 NUMBER(...)
  • 跳过空白后继续扫描

当前还不做

  • +、-、*、/
  • 标识符
  • 关键字 let、print
  • 括号、分号、等号
  • 复杂错误处理

最小扫描流程

读取输入
↓
跳过空白
↓
如果当前字符是数字
  持续向后读，直到不是数字
  输出 NUMBER(...)
↓
继续扫描后面的字符

验证样例

  • "123" → NUMBER(123)
  • "7 42 900" → NUMBER(7), NUMBER(42), NUMBER(900)
  • "12 34" → 应该正确跳过多个空格

最常见的 3 个错误

  • 只读了一个数字字符，没有读完整个数字片段
  • 只能跳过一个空格，不能连续跳过多个空格
  • 输出一个 NUMBER 后，没有继续向后扫描
