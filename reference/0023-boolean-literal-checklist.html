Reference 0023

布尔字面量清单

用途：快速确认语言是否已经支持 true / false 作为一级表达式字面量。

本课后应做到

  • scanner 能识别 true / false
  • parser 能把它们解析为布尔字面量表达式
  • evaluator 返回 Value::Boolean
  • 布尔字面量可以打印，也可以进入环境

当前仍不要求

  • match
  • if
  • 逻辑运算 && / ||
  • 布尔与整数的隐式转换

最小验证样例

print true;
print false;
let x = true;
print x;

目标输出：
true
false
true

最容易犯的错误

  • 只在 evaluator 中支持布尔值，但 scanner / parser 仍把 true / false 当普通标识符
  • 布尔字面量能求值，但不能进入环境或打印
  • 一边补字面量，一边顺手开始做 match / if，把课题做大
