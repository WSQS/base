Reference 0039

list 字面量 evaluator 清单

用途：快速确认 evaluator 是否已支持 list 运行时值和 print 输出。

本课后应做到

  • Value::List(Vec<Value>) 已加入
  • eval_expr(Expr::List(...)) 递归求值
  • print 已支持输出 list
  • 新增 evaluator 测试并通过

当前仍不要求

  • list 操作 builtin
  • list pattern 匹配

list 求值速查

Expr::List(elements) => {
    let values: Vec<Value> = elements.iter()
        .map(|e| eval_expr(e, env))
        .collect();
    Value::List(values)
}

最小验证样例

输入:  let x = [1, 2]; print(x);
期望:  输出 [1, 2]

输入:  let x = []; print(x);
期望:  输出 []

最容易犯的错误

  • list 元素没有递归求值就直接传入
  • 加了 Value::List 但忘记在 print 中加输出分支
  • 一边做 evaluator，一边顺手加 list 操作 builtin，把课题做大
