Reference 0017

左结合表达式树清单

用途：在表达式链已经能 parse 的基础上，快速确认它们是否按更自然的左结合方式构造树形。

本课后应做到

  • 加减链按左结合生长
  • 乘除链按左结合生长
  • 优先级仍然正确
  • 括号分组仍然有效

当前仍不要求

  • 比较运算
  • 赋值表达式
  • 新语句系统
  • 更高阶段复杂语法

最小思路

current = 先解析第一个左边

while 后面还是这一层负责的操作符:
  读操作符
  解析一个右边
  current = Binary(current, op, right)

返回 current

验证样例

  • print 10 - 3 - 2; → 输出 5
  • print 20 / 5 / 2; → 输出 2
  • print 1 + 2 * 3 + 4; → 输出 11

最容易犯的 3 个错误

  • 仍然把整条剩余表达式递归到右边，导致右结合树
  • 没有把当前累计结果重新当作左边继续拼接
  • 修好了左结合，却把优先级或括号行为弄��了
