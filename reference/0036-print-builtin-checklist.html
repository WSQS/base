print builtin 清单

用途：快速确认 print 已作为 builtin 函数注入且表达式语句已改为纯求值。

本课后应做到

  • print_builtin 已实现
  • print 已在 eval_program() 中注入
  • Stmt::Expr 已改为纯求值（不再自动打印）
  • evaluator 中 unreachable pattern 已清理
  • 测试通过

print builtin 速查

fn print_builtin(args: Vec<Value>) -> Value {
    match &args[0] {
        Value::Integer(i) => print!("{i}\n"),
        Value::Boolean(b) => print!("{b}\n"),
        Value::String(s) => print!("{s}\n"),
        _ => print!("{:?}\n", args[0]),
    }
    args[0].clone()
}

Stmt::Expr 速查

// 旧 → 新
Stmt::Expr { expr } => {
    eval_expr(expr, &env);  // 只求值，不输出
}

环境注入速查

pub fn eval_program(program: &Program) {
    let mut env = HashMap::new();
    env.insert("print".to_string(), Value::BuiltinFn(print_builtin));
    eval_program_with_env(program, &mut env);
}

最小验证样例

输入:  print("hello");
期望:  输出 hello

输入:  let x = 42; print(x);
期望:  输出 42

输入:  1 + 2;
期望:  无输出（表达式语句不自动打印）
