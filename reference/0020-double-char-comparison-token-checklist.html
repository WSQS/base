Reference 0020

双字符比较 token 清单

用途：快速确认 scanner 是否已经支持双字符比较运算符。

本课后应做到

  • Token 中新增 ==、!=、<=、>= 对应变体
  • scan() 能通过前瞻识别双字符比较运算符
  • 新增 scanner 测试并通过

当前仍不要求

  • parser 理解比较表达式
  • evaluator 计算布尔值
  • 条件语句

最小验证样例

let tokens = scan("1 == 2 != 3 <= 4 >= 5");

目标 token 序列：
[Number(1), EqualEqual, Number(2), BangEqual, Number(3), LessEqual, Number(4), GreaterEqual, Number(5)]

最容易犯的错误

  • 只识别了 == 和 !=，忘了同步处理 <= 和 >=
  • 前瞻后没有正确推进扫描位置，导致字符被重复消费或漏掉
  • 一边改 scanner，一边顺手去改 parser / evaluator，把课题做大
