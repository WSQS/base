Reference 0003

单字符 token 扩展清单

用途：在现有扫描器上增量添加单字符运算符 token 时，快速检查你忘了哪一步。

当前 token 种类

已支持

  • Number(i64)
  • Plus（本课新增）

后续再做

  • Minus
  • Star
  • Slash
  • 括号、等号、分号
  • 标识符、关键字

增量添加单字符 token 的标准步骤

1. 在 enum Token 里加新变体（如 Plus）
2. 在扫描循环里，加一个新的字符匹配分支
3. 遇到新字符时，先结束当前正在累计的 NUMBER
4. 推入新 token
5. 重置累计状态

必须处理的细节

  • 新 token 前后的空白应该被正确跳过
  • 新 token 之前的 NUMBER 不能被丢���
  • 新 token 之后的 NUMBER 不能丢失
  • 连续出现多个 Plus 时，每个都应输出独立 token

验证样例

  • "12 + 34" → Number(12), Plus, Number(34)
  • "7+42" → Number(7), Plus, Number(42)
  • "7 + 42 + 900" → 5 个 token，顺序正确
