Reference 0022

比较表达式求值清单

用途：快速确认 evaluator 是否已经支持比较表达式，并返回布尔值。

本课后应做到

  • eval_expr() 已支持 <、>、<=、>=、==、!=
  • 整数比较返回 Value::Boolean
  • 新增 evaluator / 程序级测试并通过

当前仍不要求

  • 布尔字面量 true / false
  • 逻辑运算 && / ||
  • if / 条件语句

最小验证样例

print 1 + 2 < 4;
print 2 * 3 == 6;
print 5 != 5;

目标输出：
true
true
false

最容易犯的错误

  • 比较表达式仍然返回 Value::Integer，而不是 Value::Boolean
  • 把比较和算术分支混在一起，导致返回类型不清楚
  • 一边改 evaluator，一边顺手加入布尔字面量或条件语句，把课题做大
