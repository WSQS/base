Reference 0004

更多单字符运算符清单

用途：在现有 `Number` + `Plus` 扫描器上继续加入 `Minus`、`Star`、`Slash` 时，快速检查你是否漏了关键步骤。

本课后应支持

  • Number(i64)
  • Plus
  • Minus
  • Star
  • Slash

当前仍不做

  • 括号
  • 等号、分号
  • 标识符
  • 关键字
  • 复杂错误恢复

每增加一个单字符 token 的固定流程

1. 在 enum Token 中添加新变体
2. 在扫描循环中匹配新字符
3. 先结束当前正在累计的 Number
4. 推入对应 token
5. 继续向后扫描

最容易重复的逻辑

  • 遇到分隔或运算符时，把当前 Number 推入结果
  • 把 value 重置为 0
  • 把 has_value 重置为 false

如果这三步在多个分支里反复出现，说明你已经接近下一次小重构时机。

验证样例

  • "12 - 3 * 4 / 2" → 按顺序输出 7 个 token
  • "7+8*9-2/1" → 无空格也能正确切分
  • "10 / 2 + 3" → 混合旧 token 与新 token 时顺序正确
