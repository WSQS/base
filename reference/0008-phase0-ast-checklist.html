Reference 0008

Phase 0 AST 清单

用途：在真正实现 parser 前，快速检查你定义的 AST 是否已经足够表达 Phase 0 的最小语句语言。

程序层

  • Program
  • 内部包含多条语句，例如 Vec<Stmt>

语句层

  • Let
  • Print

表达式层最低需求

  • Number
  • Ident
  • Binary
  • Grouping

最小结构关系

Program
  └── Vec<Stmt>

Stmt
  ├── Let { name, expr }
  └── Print { expr }

Expr
  ├── Number(...)
  ├── Ident(...)
  ├── Binary { left, op, right }
  └── Grouping(...)

验证样例

  • let x = 1 + 2; 能表示成一条 Let 语句，右侧是 Binary 表达式
  • print x; 能表示成一条 Print 语句，内部是 Ident 表达式
  • 一个程序可以包含多条语句，而不只是一个表达式

最容易犯的 3 个错误

  • 把 print 错误设计成表达式而不是语句
  • 没有 Program 层，导致只能表达单条语句
  • 在 AST 阶段一次性塞入过多未来特性，超出 Phase 0 边界
