Reference 0028

字符串字面量 parser 清单

用途：快速确认 parser / AST 是否已经支持最小字符串字面量。

本课后应做到

  • Expr 中新增了字符串字面量节点
  • parse_primary() 已支持 Token::String(...)
  • 新增 parser / AST 测试并通过

当前仍不要求

  • evaluator 支持字符串
  • Value::String
  • 字符串运算
  • 字符串 comparison
  • 字符串 pattern

最小验证样例

输入:
print "hello";

目标：
program.stmts[0] = Stmt::Print {
  expr: Expr::String("hello")
}
输入:
let name = "world";

目标：
program.stmts[0] = Stmt::Let {
  name: "name",
  expr: Expr::String("world")
}

最容易犯的错误

  • scanner 已有 Token::String，但 parser 忘记加对应分支
  • 把字符串 token 偷懒映射成 Expr::Ident
  • 一边做 parser，一边顺手去改 evaluator，把课题做大
