Reference 0005

语句基础符号清单

用途：在现有扫描器上继续加入 (、)、=、; 时，快速检查你是否漏掉了某个 token 或某个测试场景。

本课后应支持

  • LParen → (
  • RParen → )
  • Equal → =
  • Semicolon → ;

当前仍不做

  • 标识符
  • 关键字 let / print
  • 多字符运算符
  • 语法规则

固定扩展流程

1. 在 enum Token 中加入新变体
2. 在扫描循环中匹配新字符
3. 先结束当前 Number
4. 推入新 token
5. 继续扫描后续字符

验证样例

  • "(12 + 3) = 15;" → 8 个 token，顺序完整
  • "(7+8);" → 无空格也能正确切分
  • "(1+2)*(3-4);" → 新旧 token 混合时都能保持顺序

最容易遗漏的点

  • 新增分支时忘了先收尾当前 Number
  • 加了 enum 变体，但没加扫描分支
  • 有空格输入能过，但无空格输入没测到
