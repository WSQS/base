Reference 0016

表达式链解析清单

用途：在分层 parser 第一次支持连续同层表达式时，快速确认每一层是否已经能持续吸收自己负责的操作符。

本课后应做到

  • 加减层能连续处理 + -
  • 乘除层能连续处理 * /
  • 优先级仍然正确
  • 括号分组仍然有效

当前仍不要求

  • 比较运算
  • 赋值表达式
  • 完整 Pratt parser
  • 新语句种类

最小思路

parse_add_sub()
  先拿一个 mul/div 结果
  只要后面还是 + 或 -
    就继续拿下一个 mul/div 结果
    并把当前结果更新为新的左边

parse_mul_div()
  先拿一个 primary
  只要后面还是 * 或 /
    就继续拿下一个 primary
    并把当前结果更新为新的左边

验证样例

  • print 1 + 2 + 3; → 输出 6
  • print 2 * 3 * 4; → 输出 24
  • print 1 + 2 * 3 + 4; → 输出 11

最容易犯的 3 个错误

  • 某一层只拼一个 Binary 就返回，导致链��表达式被截断
  • 忘了把新拼出的结果重新作为左边继续处理
  • 修好了链式表达式，却把括号或优先级关系弄坏了
