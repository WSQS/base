Reference 0037

list token scanner 清单

用途：快速确认 scanner 已经识别 [ ] 为独立 token。

本课后应做到

  • Token::LBracket 和 Token::RBracket 已加入
  • scanner 已支持 [ 和 ]
  • 新增 scanner 测试并通过

当前仍不要求

  • parser 支持 list 字面量
  • AST 支持 list 表达式
  • evaluator 支持 list 值

命名对照

{ } = LBrace  / RBrace   (match 体、fn 体)
[ ] = LBracket / RBracket (list 字面量)
( ) = LParen  / RParen   (分组、参数列表)

最小验证样例

输入:
[1, 2, 3]

目标:
[LBracket, Number(1), Comma, Number(2), Comma, Number(3), RBracket]

最容易犯的错误

  • 把 ] 当成 } 的笔误
  • 忘记先 flush 再 push 方括号 token
  • 一边做 scanner，一边顺手改 parser，把课题做大
