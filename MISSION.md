# Mission: 实现自己的编程语言

## Why
我希望尝试自己实现一些有趣的编程语言，并通过亲手构建语言来真正理解它们是如何工作的。

## Success looks like
- 先独立实现一个最小但完整可运行的语言闭环
- 能解释并亲手实现词法分析、语法分析、AST、求值/执行这几个核心环节
- 能在这个最小闭环基础上继续扩展语言能力，逐步走向更强的实验平台
- 长期尝试让语言具备自举的演化路径，并进一步探索 LLVM 后端

## Constraints
- 目前学过一点词法/语法分析
- 希望教学风格理论和实践并重
- 近期优先完成 Phase 0：最小可运行闭环，而不是过早追求完整自举能力
- 当前 Phase 0 的语言边界是：数字、标识符、`let`、`print`、`+ - * /`、`(`、`)`、`=`、`;`
- 当前 Phase 0 的结构路线是：lexer -> parser -> AST -> evaluator
- 当前 Phase 0 的第一批语句是：`let name = expr;` 与 `print expr;`

## Out of scope
- 现在先不追求一步到位实现“自举语言所需的全部能力”
- 现在先不追求工业级优化器、JIT 或完整编译器工具链
- 现在先不追求设计一门大而全的通用编程语言
- 现在先不追求在 Phase 0 之前提前引入 `if`、`while`、函数、复杂数据结构等更高阶段能力
