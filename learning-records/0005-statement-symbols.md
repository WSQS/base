# Learning Record 0005: statement symbols

## Context
用户已完成 Lesson 0005，主题是在现有扫描器上继续加入 `(`、`)`、`=`、`;` 四种基础语句符号。

## What was learned
- 用户已能持续扩展单字符 token 集合，并保持数字与运算符、分隔符混排时的正确切分顺序。
- 用户已通过测试验证括号、等号、分号加入后，输入已经开始接近真实语句，如 `(12+3)=15;`。
- 用户已建立“先完成 token 切分，再考虑更高层语义”的实现节奏。
- 当前实现中的重复 `flush` 逻辑仍被保留，作为后续结构整理的明确切入点。

## Why it matters
完成这一步后，扫描器已经能支撑更像程序语句的输入，而不再只是简单表达式片段。下一步引入标识符和关键字时，用户会首次从“符号语言”进入“名字语言”。

## Implications for next lessons
- 下一课最自然进入标识符扫描。
- 在引入标识符之前，可以先做一次小范围 code review 优化，处理当前扫描器中的重复收尾逻辑。
- 标识符课应继续保持实践驱动，并通过输入/输出与测试验证名字切分行为。
